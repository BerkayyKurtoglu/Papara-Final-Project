# Recipe-Quest
- RecipeQuest is a phenomenal application that allows you to explore dishes from all cuisines, save your favorites, and more. No internet? No problem! It can display up to fifty recipes you visited even while offline. Additionally, it suggests recipes similar to your favorites when your phone is idle. It's beyond perfect! ðŸ˜‰.
